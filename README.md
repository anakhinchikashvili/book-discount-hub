# 📚 Book Marketplace

A **multi-publisher online bookstore** developed as a Bachelor's thesis.

The platform enables multiple independent publishers to manage their own books, inventory, pricing, and orders within a single marketplace while providing customers with a unified shopping experience.

---

## ✨ Features

- 📖 Multi-publisher marketplace
- 🔐 JWT authentication
- 👥 Role-Based Access Control (RBAC)
- 🔍 Book search & filtering
- 🛒 Shopping cart & wishlist
- 🔄 Cross-Vendor Checkout
- 📦 Order management
- ✅ Publisher approval workflow
- ⚡ Optimistic locking for inventory consistency

---

## 🛠️ Tech Stack

### 💻 Backend
- Java
- Spring Boot
- PostgreSQL
- Spring Security
- JWT

### 🎨 Frontend
- React
- Vite

### ☁️ Deployment
- AWS EC2
- Nginx
